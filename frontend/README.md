# Hierarchy Processor API

A full-stack web application for processing and managing hierarchical data structures.

##  Features

* Backend API built with Node.js and Express
* Frontend built with React + Vite
* REST API integration
* Hierarchical data processing
* Clean project structure
* Responsive UI

##  Installation

### Clone Repository

```bash
git clone https://github.com/hemanthreddy1111/Hierarchy-Processor-API.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
